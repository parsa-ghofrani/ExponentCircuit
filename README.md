# Exponent Circuit
This program (`mainFile.py`) is for testing the actual circuit that computes $$a^b$$ in binary format.

## Already tested components:
1. `Adder`
2. `Subtractor`
3. `SubPos`
4. `IntSqrt`
5. `FracSqrt`
6. `FloatSqrt`
7. `IntMul`
8. `FloatMul`
9. `FloatSquare`
